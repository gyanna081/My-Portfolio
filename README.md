# Geanna Portfolio

A clean student portfolio layout built with Tailwind CSS.

## Files

- `index.html` — portfolio page with a two-column layout, profile card, and experience section.

## Usage

1. Open `index.html` in your browser.
2. Edit the profile, contact details, and experience items as needed.

## Notes

- The layout uses a modern professional font and badge-style headings.
- The left column is a fixed profile card with rounded corners.
- The right column uses vertical spacing (`space-y-8`) for experience sections.
